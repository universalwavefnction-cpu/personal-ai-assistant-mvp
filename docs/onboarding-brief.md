# Post-Payment Onboarding Brief

Send this after payment, not on the landing page.

```text
Спасибо. Чтобы собрать первую версию ассистента за 2 дня, пришлите, пожалуйста:

1. Ваше имя и Telegram.
2. Для чего ассистент нужен в первую очередь?
3. Три главных сценария:
   - например: разбирать голосовые;
   - писать посты;
   - отвечать по заметкам;
   - объяснять PDF;
   - планировать день.
4. До 10 материалов:
   - заметки;
   - PDF;
   - статьи;
   - конспекты;
   - примеры текстов;
   - инструкции;
   - ссылки.
5. Какой тон нужен?
   - коротко / подробно;
   - мягко / прямо;
   - формально / живым языком.
6. Один пример хорошего ответа в вашем стиле.
7. Один пример плохого ответа, как не надо.
8. Нужны ли голосовые?
9. Есть ли уже VPS/сервер?
10. Есть ли предпочтительный AI-провайдер или API-ключ?
11. Подтверждение:
   - расход AI-модели считается отдельно по токенам после включенного демо-лимита;
   - ассистент не принимает важные решения вместо вас;
   - ассистент не отправляет сообщения от вашего имени без подтверждения.
```

## Short version

```text
Минимум для старта:

1. 3 задачи, где ассистент должен помогать.
2. До 10 файлов/заметок.
3. Какой тон ответа вам нужен.
4. Сервер: есть или нужна инструкция.
```

## Example scenarios to offer

- Ask questions across my notes and files.
- Turn voice notes into structured notes.
- Draft posts/emails/messages in my style.
- Explain PDFs and articles simply.
- Create daily/weekly plans.
- Help think through ideas by asking questions.
- Translate with meaning and tone.
- Prepare replies to difficult messages.
